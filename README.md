# Application
## About
## Install
1. Install project 

```
npm install
or 
yarn install
```

2. Reference project

```
react-native link
```

#### Noted: please make sure react-native-cli installed

```
npm install -g react-native-cli
```

## Run 
### Android

1. Install Genymotion 
2. Link Genymotion SDK to Adnroid SDK
3. Start emulation
4. Run debug

```
react-native run-android
```

### IOS

1. Open rootApp/ios/mycat.xcodeproj
2. Run debug

## Debug
1. Install react-native-debuger
2. Open debuger
3. Start debug mode

```
Android : control + M
IOS : command + D
=> Debug JS Remotely
```
